# Backup e Restore de Pastas

## Instalação
brew install gnupg

## Baixar script
git clone https://github.com/lucas-santos-dev/backup-restore-folders.git

## Dar permissão de execução
chmod +x backup.sh restore.sh

## Executar script
./backup.sh <pasta_origem>

./restore.sh <nome_do_arquivo>.tar.gz.gpg
